Abstract:

Para este trabalho de casa foi proposta a criação uma API de dados em NodeJS capaz de receber apenas pedidos GET e de responder com os dados pedidos.Em primeiro lugar, foi necessária a passagem de um dataset em xml para formato json de maneira a permitir a inserção dos dados na base de dados em mongo.

O servidor criado corre na porta 3012 e responde a pedidos com a seguinte formatação de exemplo: localhost:3012/obras.A API permite obter a lista de obras de um determinado período,de um determinado ano de criação ou de um determinado utilizador, tudo isto utilizando query params.

Por outro lado pode-se também obter informações relativas ao compositores das obras, utilizando a seguinte formatação : localhost:3012/compositores. Neste caso é possível obter a listagem de todos os compositores ou então obter todas as obras de um determinado compositor.
